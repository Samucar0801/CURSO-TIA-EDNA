Tag Header

Marca o cabeçalho do site (banner), onde geralmente estão presentes a marca, navegação do site e as vezes uma informação introdutória.

Tag OL, UL, DL, e Li

A Tag UL é ultilizada para criar lista não ordenadas (unordered list). Ex:

  • Menus de um site
  • lista de tópicos mais visitados
  • Lista de ingredientes

OL é uma lista ORDENADA (ordered list). Ex:

Top 5 produtos mais vendidos do mês:

  1. Barbeador Elétrico
  2. Fritadeira S Óleo
  3. Carrinho de Controle Remoto
  4. Mouse Dragon Verde
  5. Banco Imobiliario

Tag DL

a tag DL marca uma lista de descrição (description list) A lista é criada com dois elementos DT e DD, Ex:

A tag DT
Esta tag marca o topico a ser falado
A tag DD
Esta tag marca a descrição do topico falado

Tag Section

semanticas para usar no texto

Blockquote
ultilizado para fazer citaçoes no HTML. possui um atribulto chamado "cite", onde Colocamos nele o valor: onde retiramos a citação
Q
Marca uma citação inline, ou seja, dentro do texto.
cite
Tag que serve para marcar obras livros, filmes, pinturas, etc...
EM
Tag ultilizada para dar enfase em uma parte do texto
strong
utilizado para marcar uma parte importante do site, a deixa em

A tag aside é utilizada para marcar conteúdo complementar ao principal do site.

Exemplos: Lista de posts mais visitados, informações extras como a descrição de um termo usado no artigo e mais.
